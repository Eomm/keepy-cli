Usage: keepy init [--yes|-Y] [--overwrite|-F]
  create a keepy-store.json to save keys

  -Y, --yes
      Init the keepy-store.json with a random password printed out

  -F, --overwrite
      Overwrite keepy-store.json if it exists

  -w, --password <string>
      Password to use instead of generated one when --yes is used

  -h, --help
      Show this help message
