Usage: keepy delete [--env|-e] [--key|-k <string>] [--tag|-t <string>] [--password|-w <string>] [--help|-h]
  delete keys stored in keepy-store.json

  -k, --key <string>
      The key to delete

  -t, --tag <string>
      The tag that keys must have to be deleted
  
  -w, --password <string>
      Password to use instead of prompt

  -h, --help
      Show this help message
